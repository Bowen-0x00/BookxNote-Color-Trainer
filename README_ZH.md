# BookxNote 快捷换色器 🎨

简体中文 | [English](README.md)

![软件界面](images/app.png)

这是一个轻量级的开源工具，允许你通过全局快捷键（`F1`-`F4`）在 **BookxNote Pro** 中瞬间切换画笔颜色。它通过直接修改软件内存实现换色，让你在沉浸式阅读和做笔记时，彻底告别繁琐的鼠标点击。

### ⚠️ 版本兼容性说明
基于内存地址的修改对软件版本极其敏感。本项目的基址与指针链专门针对以下版本逆向提取并测试通过：
- **BookxNote Pro 版本:** `3.0.0.2023 (64位)`

*注意：如果软件后续发生升级，内存结构可能会改变，导致程序失效。届时需要重新使用 Cheat Engine 扫描并更新源码中的 `OFFSETS` 列表。*

### ✨ 核心功能
- **全局快捷键控制:** 真正的不打断心流，按下快捷键瞬间换色，焦点始终留在你的笔记上。
- **网页级颜色配置:** 告别晦涩的内存数值，直接使用 `#FF5733` 这种最常见的十六进制颜色代码配置你的专属调色板。

### 🧠 原理解析
本项目基于**内存读写（Memory Editing）** 技术实现。
通过逆向分析，我们找出了 BookxNote Pro 内部控制画笔颜色的多级指针链。Python 脚本利用 `pymem` 库顺藤摸瓜，实时定位到动态变化的真实内存地址。

同时，程序内置了颜色转换算法：它会自动接收你输入的常规 RGB 颜色代码，将其翻倍扩充，并按照 `蓝(B)-绿(G)-红(R)-透明度(F)` 的顺序重新组装成软件底层所需的 64 位长整型格式（`BBGGRRFF`），最后直接写入内存实现瞬间换色。

### 🚀 如何使用

1. 安装依赖库：
   ```bash
   pip install pymem keyboard
   ```
2. 打开 **BookxNote Pro** 软件。
3. 运行 `src` 目录下的脚本：
   ```bash
   python src/BookxNote_Color_Trainer.py
   ```
4. 在软件中按下 `F1`、`F2`、`F3` 或 `F4` 即可瞬间切换画笔颜色！

### 🛠️ 自定义颜色
你可以通过修改代码中的 `COLORS` 字典来绑定你专属的颜色。只需填入标准的十六进制颜色代码即可：

```python
COLORS = {
    "F1": hex_to_8byte_color("#59C6FF"), # 把这里的代码换成你喜欢的颜色！
    # ...
}
```
